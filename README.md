This repo contains
FreeSWITCH binaries and module config
for a generic FreeSWITCH-based PBX.

Most module config
and all directory/dialplan config
is via `mod_xml_curl`.

The binaries and module config
support the following.

- Video calls
- Conference calls
- Loopback calls
- Voicemail
- Music on hold
- Auto-attendant
- WebRTC (SIP or Verto)
- CDRs
