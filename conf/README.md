This repo contains
FreeSWITCH binaries and module config
for a simple FreeSWITCH-based SIP PBX.
Configuration other than module auto-load config
is via `mod_xml_curl`.

The binaries and module config
support the following.

- Video calls
- Conference calls
- Loopback calls
- Voicemail
- Music on hold
- Auto-attendant
- CDRs
- WebRTC
